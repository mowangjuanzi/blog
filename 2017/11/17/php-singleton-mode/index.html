好久没发表文章了。今天说一下单例模式。

单例模式主要有以下几个关键点：

  • 对象应该可以被系统中的任何对象使用。
  • 对象不应该被存储在会被覆写的全局变量中。
  • 系统中不应超过一个要单例的对象。
